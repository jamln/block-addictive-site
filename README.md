# Block Addictive Site

A Chrome extension that prevent interesting site distract myself

**Lightweight URL Blocker Extension**
No login required, simple and easy to use
This extension helps you easily block distracting websites,  protecting your productivity and attention. No registration or login required, just install and use.

**Supports time period settings**
You can customize the time periods during which to block specific websites, such as during work or study hours.

**轻量级网址拦截插件**
无需登录，简单易用
本插件帮助您轻松地拦截分散你注意力的网址，保护您的生产力和注意力。无需注册登录，直接安装即可使用。

**支持时间段设置**
您可以自定义拦截的时间段，例如在工作时间或学习时间内拦截特定网址。
